# ZiGate Commands

The purpose is to document the ZiGate commands:

* Sequence: what is expected when the command is sent
* Ack:
* SQN:
* Layer: Zigbee layer ( ZDP, ZCL), ZIGATE indicated that is a ZiGate proprietarory command
* 8012:
* Command: short description of the command
* NwkId 2nd Bytes: True is the 2nd byte is a Network Short id ( most of the ZCL commands)
